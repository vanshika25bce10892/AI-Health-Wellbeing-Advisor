# AI-Health-Wellbeing-Advisor
Overview
The AI Health and Well Being Advisor is a straightforward AI Technology Project that assists users in determining basic health information through an interactive program
The two small tool projects combine into one system:
	1.	Body Mass Index (BMI) and Diet Advisor calculates Body Mass Index (BMI) and gives general dietary recommendations.
	2.	Symptom Checker - asks about various common symptoms to obtain possible conditions and such basic advice.

This is a demonstration of Artificial Intelligence skills, like simple rules and decision making to help people solve common health problems.
This project is for people who want to learn about the basics of Artificial Intelligence.

Features:

•	The program can calculate a persons Body Mass Index based on what they tell us
•	It can tell someone if their Body Mass Index is too low, right too high or very high
•	It gives people some basic ideas for eating healthy and exercising
•	The program can help figure out if someone might have a health problem based on their symptoms
•	It uses simple rules to try  determine what might be wrong with someone based on their symptoms
•	It is easy to use by typing commands into the computer

Concepts in Artificial Intelligence

This project uses  basic concepts in Artificial Intelligence:
•	Rule-Based System

The program follows rules to give people advice based on their Body Mass Index.
•	Decision Tree Logic

The program looks at all the symptoms someone tells us about and tries to figure out what might be wrong with them.
•	Basic Intelligent Agent Interaction

The program talks, to people. Gives them advice based on what they tell us.
