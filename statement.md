AI Health & Wellbeing Advisor – Project Statement

## Problem Statement

In our lives we often have small health worries but we don't know where to find reliable help. Many people search the internet, which can be confusing or give information. Others ignore signs because they are not aware. We need an trustworthy tool that gives basic health information in an easy and interactive way. It should not confuse users. Replace doctors advice.

## Scope of the Project

This project is a beginner- AI system that provides basic health awareness. It works in an limited way by:

* Calculating Body Mass Index (BMI) using user inputs

* Explaining BMI results in a way

* Suggesting general diet and lifestyle improvements

* Taking user symptoms. Offering possible explanations

* Giving basic tips for better health awareness

This system is not a substitute for doctors or medical professionals. Its purpose is to inform, not diagnose.

## Target Users

This project is mainly for:

* Students learning Artificial Intelligence who want an example

* Beginners curious about AI in life


## Features at a high level
BMI Insight Tool - To calculate BMI and provide the user with an understanding of how their body is doing.

Simple Health Suggestions - To present simple health suggestions to users.

Symptom-Based Guidance - To show the user different possible reasons based on what they enter.

Smart Rule Logic - To use simple AI rules for making decisions and suggestions.

Interactive Experience - To provide users with an interactive experience using a simple text-based interface.

Learning-Oriented Design - To teach the user basic AI concepts (rule-based systems and decision-making) in practical terms.
